# StriaGO

**StriaGO** — Auto-farm script for **Build A Ring Farm** on Roblox.

> Built with performance in mind. No bloat, no keys, no obfuscation.

---

## Features

| Feature | Description |
|---|---|
| **Auto Harvest** | Automatically collects harvest boxes and mature crops |
| **Auto Collect** | Gathers all collectible items in the farm |
| **Auto Sell** | Sells produce at the sell shop |
| **Auto Plant** | Automatically plants seeds on available plots |
| **Auto Replant** | Replants high-tier (Celestial, Eternal) seeds |
| **Auto Upgrade** | Upgrades Seed Luck, Saw Yield, Sprinkler, and more |
| **Auto Pet Collect** | Collects pet eggs and pet rewards automatically |
| **Event Detection** | Detects active weather/events (Rain, Galaxy, UFO, etc.) |
| **Auto Event Claim** | Automatically claims event rewards |
| **Code Redeemer** | Redeems all active codes with one click |

## Usage

### Loader (one-liner)

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dovanedo00/StriaGO/master/StriaGO.lua"))()
```

### Direct execution

Copy the contents of `StriaGO.lua` into your executor and run.

### Controls

| Key | Action |
|---|---|
| **RightShift** | Toggle GUI open/close |
| Toggle switches | Enable/disable specific auto features |
| **Redeem All Codes** button | Redeem all active codes |
